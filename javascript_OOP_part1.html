class

  • Class is Program template that provides data and logic
  • class is used as Blue Print to create multiple Instances

    • Class Configuration -

  • Class are config using 2 Technique
  • Declaration - 
                class Name{

                // Members

            }
  • Expression - 
                const Name = class{

            }

    • Class Members

  • Property
  • Accessor
  • Methods
  • constructor

    • Property

  • A Property is responsible for handeling data in class
  • config any type of data in into property
  • In Es6+ we can make any Property Static - Shared Value in Objects which stays long as Program run.

    • _underscore , #hash - prefix in Property

  • _underscore is used to Mark as internal use but accessible
  • #hash forces to access only internally

    • Accessor

  • Accessor are used to control over Property
  • used to handeling Read and Write
  • Accessor are classified into 2 types
    1. get - Getter
    2. set - Setter
