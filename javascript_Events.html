What is Event Progatation and Bubbling

  • Progatation is Mechanism where child simulate the event on parent
  • we can prevent using "stopProgatation()" in Child

    • How to prevent Generic elements

  • Generic elements are pre-defined functions whit automatic calling function on each request
  • we can prevent using "preventDefault();"

    • What is Event Loop

  • Event loop is Mechanism of executing task provided to an event
  • it follows standard Execution Order
    1. General Task
    2. Micro Task
    3. Macro Task

    What is Event Profiling

  • Event Profiling is technique where deveoper can track activity and time took to execute
