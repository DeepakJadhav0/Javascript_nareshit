JavaScript Input Techniques

Input is the process of accepting data from the user in different ways.

  1. Query String

    Input is passed using the URL. You can access it via window.location.search.

    Category:
    [waiting for input]
    Product:
    [waiting for input]
    Brand:
    [waiting for input]
    // Example URL:
// page.html?category=Phones&product=Galaxy&brand=Samsung

let params = new URLSearchParams(window.location.search);
document.getElementById("lblCategory").textContent = params.get("category");
document.getElementById("lblProduct").textContent = params.get("product");
document.getElementById("lblBrand").textContent = params.get("brand");

  2. Prompt

    The prompt() method displays a dialog to take user input directly.

    let name = prompt("Enter your name:");
console.log("User entered: " + name);
