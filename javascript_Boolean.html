Boolean & Operators

In JavaScript, a Boolean is a primitive data type that can only have two values: true or false. It is commonly used in conditional statements, comparisons, and logic to control the flow of the program based on certain conditions. 

        let a = True;
        let b = False;


        console.log(True)   // True

Operators

Operators are used to perform operations on variable and values

Arithemitic Operators -


Arithemitic are used to perform operations like Addition , substraction , multiplication etc


        let num1 = 10;
        let num2 = 20:

        num1 + num2     // 30
        num2 - num1     // 10
        num1 * num2     // 200
        num1 / num2     // 0.2
        num1 % num2     // 10
        num1 ** num2    // 10 of power 20


Assignment Operators -


Assignment operations are used to assign value to variable


        let num1 = 10;
        let num2 = 20:

        num1 += 10      //num1 = num1 + num1
        num1 -= 10      //num1 = num1 - 10
        num1 *= 10      //num1 = num1 * 10
        num1 /= 10      // num1 = num1 / 10

comparisons Operators -


comparisons operations are used to compare values , return boolean value


        let num1 = 10;
        let num2 = 20:

        num1 > num2     //False
        num1 < num2     //True
        num1 >= num2    //False
        num1 == num2    //False
        num1 === num2   //False
        num1 !== num2    //True

Logical Operators -


Logical operators are used to combine or invert boolean values.


        let num1 = True;
        let num2 = False:

        num1 && num2     //False
        num1 || num2    //True
        !num1           //False

Ternary Operators -


Ternary operator is Shorthand for If-else
