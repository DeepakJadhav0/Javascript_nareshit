Variable

Variables are storage locations where you store the data and use it.

  1. Declaration
  2. Initialization
  3. Assignment
Syntax - keyword variable_name;

Variables are declared using keywords.

Var

It configures a function-scoped variable. You can declare it in any block of a function and access it from any other block in the same function.

Shadowing - It is the process of re-declaring a variable with the same name within the same scope.

e.g -
    var x;        // Declaration
    x = 10;       // Initialization
    var x = 20;   // Shadowing

Hoisting - It's the mechanism of accessing a variable from any location regardless of order.

e.g -
    x = 10;  // Initialization
    var x;   // Hoisting

Let

It configures a block-scoped variable. It is accessible within the block and inner blocks.

Use when the variable needs to be reassigned.

let x;      // Declaration
x = 10;     // Assignment

Const

It configures a block-scoped variable. Cannot be reassigned.

Use when the value should not change.

const x;         // Invalid
x = 10;          // Invalid
const x = 10;    // Valid
x = 20;          // Invalid

Global Variable

A variable declared outside a function is considered global. You can also explicitly attach a variable to the global object.

window.global_var = "Deepak";

console.log(window.global_var); // Accessible anywhere in code
