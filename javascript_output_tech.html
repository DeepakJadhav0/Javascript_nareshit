JavaScript Output Techniques

Output is the process of rendering results in the UI using various JavaScript methods.

  1. alert("message")

    Displays a popup message.

    alert("Message");

  2. confirm("message")

    Displays a confirmation box. Returns true for OK, false for Cancel.

    confirm("Are you sure?");

  3. document.write("...")

    Writes directly to the document (and replaces the entire DOM).

    document.write("Hello World");

  4. textContent

    Replaces only the text inside an element — HTML tags will not be rendered.

    element.textContent = "Plain text";

    This is normal text (before textContent).

  5. innerHTML

    Renders HTML content inside an element — tags are interpreted as HTML.

    element.innerHTML = "HTML content";

    This is normal text (before innerHTML).
