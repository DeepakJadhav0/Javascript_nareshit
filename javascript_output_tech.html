Javascript Output Technique

Output is Rendering result in UI

  1. Alert("Message")

    It pops the message box

    syntax - alert("Message");

  2. Confirm("message | expression (ok - True or cancle - False)")

    Similar to alert but option for ok or cancel - Return Boolean Value

    syntax - confirm("Message return True or false")

  3. document.write("message | Expression | HTML")

    It renders output on a new screen of same page

    syntax - document.write(" This is Html")

  4. TextContent

    Used with selector - Renders Only text container - Render onlt Text

    syntax - document.querySelect('tagname').textContent = "This is TextContent with HTNL"

    This is Normal Sentence without HTML

  5. InnerHTML

    Used with Selector - Render HTML inside Container - render as a child element
