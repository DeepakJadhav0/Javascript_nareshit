String Manipulation part - 1

is used to Manipulat string

  1. length() - returns total count of char in string
  2. indexOf('char') - returns the first occurance index no. of char
  3. lastIndexOf("char") - returns the index of last occurance
  4. startsWith('char') - return boolen true or false
  5. endsWith('char') - returns boolean value


    var name = "john Snow"

    console.log(name.length())  // 9
    console.log(name.indexOf('n')) // 3
    console.log(name.lastIndexOf('n'))  // 6
    console.log(name.startsWith('j')) // True
    console.log(name.endsWith('o'))  // False

String Manipulation part - 2

is used to Manipulat string

  1. trimStart() - It removes the blank space from Startimg of string.
  2. trimEnd() - It removes the blank space from End of string.
  3. trim() - removes both side space from start and end.
  4. split() - It splits a string in specified char and returns the array.
  5. slice() - It extratct and return the substring from string.
  6. replace() & replaceAll() - both replaces the string with new char.
  7. include() - chech that string include specific sub string.


    var name = "john Snow"


    console.log(name.trim().length)   // 10
    console.log(name.split(" "))    // ['john','snow']
    console.log(name.slice(0,3))    // joh
    console.log(name.replace("john", "Deepak"))     // Deepak snow
    console.log(name.includes("snow"))    //True

String Compare

"==" and "===" are both used to compare string.

  1. == : to compare value of different Datatypes.

  2. === : Can compare only of similar datatype , its strict Equal.


      var num = "20"

      console.log(num == 20)    // True

      console.log(num === 20)     // False
